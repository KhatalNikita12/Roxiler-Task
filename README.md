# 🚀 Roxiler Transaction Dashboard

A MERN stack dashboard that visualizes transactions with filtering, search, statistics, bar chart, and pie chart. Built with **MongoDB**, **Express**, **React**, **Node.js**, **Tailwind CSS**, and **Recharts**.
## 📁 Project Structure
/transaction-dashboard
│
├── task-backend # Node.js + Express + MongoDB backend
├── task-frontend # React.js frontend with Tailwind CSS
## 🛠️ Technologies Used

- **Frontend**: React.js, Tailwind CSS, Recharts
- **Backend**: Node.js, Express.js, Mongoose
- **Database**: MongoDB or MongoDB Atlas
- **Others**: Axios, CORS, dotenv

  ### 1. ✅ Clone the Repository

git clone https://github.com/KhatalNikita12/Roxiler-Task.git
cd Roxiler-Task
# Backend
cd task-backend
npm install
# Frontend
cd ../task-frontend
npm install
# Run the Project
cd task-backend
nodemon index.js
# Frontend
cd ../task-frontend
npm start
Usage Guide
Click "Initialize Database" to load sample transactions.

Use the dropdown to select a month.

Search transactions by title, description, or price.

Navigate pages using Previous/Next.

View:

🔢 Total Sales, Sold & Unsold Items

📊 Bar Chart: Price ranges

🥧 Pie Chart: Categories
